# Full specification of the `54` command
### Polling rate command

**(0) Specification**

Byte (Index) | Explanation | Values
:---: | --- | ---
`0` | Command | `54`
`1` | Unknown | `00`
`2` | Num of Steps | See **(1)**
`3-End` | Unknown | `00`

**(1) Steps**

The polling rate value is given by
```
rate = max_value / (2 ^ (step - 1))
```

Mouse | Max Steps | Max Value
--- | :---: | :---: | :---:
Rival 310 | `4` | `1000` Hz
