# Full specification of the `53` command
### DPI command

**(0) Specification**

Byte (Index) | Explanation | Values
:---: | --- | ---
`0` | Command | `53`
`1` | Unknown | `00`
`2` | DPI ID | See **(1)**
`3` | Steps (see **(2)**) | Num of steps
`4` | Unknown | `00`
`5` | Unknown?? | `42`
`6-End` | Unknown | `00`

**(1) DPI ID**

Value | Slot
:---: | :---:
`01` | First
`02` | Second

**(2) Steps**

DPI value is given by
```
DPI = step_value + steps * step_value
```

Mouse | Step Value | Max Steps
--- | :---: | :---:
Rival 310 | `100` | `119` (`12000` DPI)
